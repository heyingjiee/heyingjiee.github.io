聚集函数 ​

当有分组时，Select只能是聚集函数、分组字段

sql
SELECT count(*) AS total,字段A 
FROM 表名
GROUP BY 字段A
1
2
3

还有一种用法，SUM(字段B)用来一个组内的字段B求和，但是B字段是性别，即man、woman

我们也可以通过WHEN只统计组内的部分信息

sql
SELECT SUM(CASE WHEN 字段B='man' THEN 1 ELSE 0 END) AS 字段B
FROM 表名
GROUP BY 字段A
